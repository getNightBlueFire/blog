Основные технологии В этой части справочной документации рассматриваются все технологии, являющиеся неотъемлемой частью Spring Framework. В первую очередь, это контейнер инверсии управления (IoC) Spring Framework. После подробного рассмотрения контейнера IoC Spring Framework следует подробное описание технологий аспектно-ориентированного программирования (АОП) Spring. Spring Framework имеет собственный фреймворк АОП, который концептуально прост для понимания и успешно удовлетворяет 80% требований АОП в корпоративном программировании на Java. Также рассматривается интеграция Spring с AspectJ (в настоящее время это самая богатая по функционалу и, безусловно, самая зрелая реализация АОП в корпоративном пространстве Java). Обработка AOT может использоваться для предварительной оптимизации вашего приложения. Обычно она применяется для развертывания нативных образов с помощью GraalVM.
